# Grama Urja ⚡

## Smart Village Power Monitoring System

Grama Urja is a real-time Flutter + Firebase mobile application designed to help rural communities monitor electricity availability across multiple villages. The app allows users to track village-wise power status, receive live updates, and visualize electricity activity through an interactive analytics dashboard.

---

# 🚀 Features

## ⚡ Real-Time Power Monitoring
- Monitor electricity availability instantly
- Live ON/OFF status updates
- Realtime synchronization using Firebase Realtime Database

---

## 🏘️ Village-wise Monitoring
- Separate monitoring for multiple villages
- Independent power status for each village
- Dynamic village selection dropdown

Supported villages:
- Kavuru
- Guntur
- Vijayawada
- Tenali

---

## 📊 Power Usage Analytics
- Interactive analytics chart using FL Chart
- Smooth curved graphs
- Visual representation of electricity activity
